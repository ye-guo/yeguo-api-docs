# ⚙️ 配置

使用 `YG-API-SDK` 的一些配置

:::warning{title=注意}

- JDK 8+
- Maven || Gradle
  :::

## 🌱 SpringBoot

### 🍋 Maven

```xml

<dependency>
    <groupId>icu.yeguo.common</groupId>
    <artifactId>yeguo-api-sdk</artifactId>
    <version>1.0.1</version>
</dependency>

```

### 🍇 Gradle

```gradle

  implementation 'icu.yeguo.common:yeguo-api-sdk:1.0.1'

```
